# 对吴恩达 Prompt Engineering 课程的评价

优点：

- 非常适合零基础入门
    
- 1~2 小时即可看完
    
- 建立 Prompt 思维方式
    

缺点：

- 内容较浅
    
- 不涉及工程化
    
- 不涉及 Agent
    
- 不涉及 RAG
    
- 不涉及 Tool Calling
    

结论：

把它当作热身课即可。

# 总结

- 《ChatGPT Prompt Engineering》概念没有过时，但只能作为热身课程。
    
- 2025.9 的新版 RAG 课程已经可以替代旧版 Advanced RAG。
    
- Agentic AI 是目前最贴近企业 Agent 开发的入门课程。
    
- vLLM 课程能够补足推理引擎与模型服务端知识。
    
- 你的学习路径本身已经覆盖了从 Prompt → Tool Calling → RAG → Agent → 企业 AI 系统的完整链路。
    

视频课程的价值主要在于：

```text
看别人做一遍
        ↓
建立直觉
        ↓
自己跟着实现
        ↓
再回到学习路径补细节
```

不要把视频当主线。

真正让你成长的是：

代码实践 + 项目落地 + 复盘总结。
