## 轨道控制器
### OrbitControls( object : [Camera](https://threejs.org/docs/index.html#api/zh/cameras/Camera), domElement : HTMLDOMElement )

[object](https://threejs.org/docs/index.html#api/zh/cameras/Camera "Camera"): （必须）将要被控制的相机。该相机不允许是其他任何对象的子级，除非该对象是场景自身。  
domElement: 用于事件监听的HTML元素。(可选)

1. 控制摄像机视角查看物体角度
```
import { OrbitControls } from 'three/examples/jsm/controls/OrbitControls'

const controls = new OrbitControls(camera, renderer.domElement)
controls.update()
```
2. 设置阻尼
```
// 使控制器具有惯性
controls.enableDamping = true
// 惯性系数
controls.dampingFactor = 0.25
// 是否允许缩放
controls.enableZoom = true
// 是否允许旋转
controls.enableRotate = true
// 是否允许平移
controls.enablePan = true
```
3.设置轨道自动旋转
```
controls.autoRotate = true // 是否自动旋转
controls.autoRotateSpeed = 10 // 自动旋转速度
```
