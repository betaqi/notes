- 概念/逻辑/物理模型、主键外键、范式

### 更实际的建议

你学 FastAPI，大概率会配合 **SQLAlchemy ORM**，这样你其实**很少手写 SQL**，框架帮你生成。

所以顺序建议是：

```
MySQL基础（1-6章） 
    → FastAPI + SQLAlchemy 做一个项目
        → 遇到性能问题再回来学索引(第14章)
            → 需要复杂查询再补子查询细节
```

**结论：先学约40%的内容，够支撑你做项目了，别让学课程耽误了实际动手。**
